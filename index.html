Welcome to Tetriso GitHub page

Tetriso is javascipt game written in HTML5 and Javascript. It uses howler.js library and jQuery framework.

Demo

Tetriso demo page.

Controls 

UP arrow - Rotate Termino
DOWN arrow - Move Termino faster down
LEFT arrow - Move Termino left
RIGHT arrow - Move Termino right
Q key - stash termino for later use

Functionality

I have several objects:

Main.js

Just run the whole application.

Tetriso.js

This class sets the propper size for drawing new terminos and cares about key and link bindings, also cares about showing informational dialogs named below and cares about muting sounds. 

- CONTROLS
- INFO
- HIGH SCORE

Playfield.js

Class which cares about drawing stuff in those three canvases and the main logic of the game is there.

PlayMatrix.js

This class represents the matrix where the game is played in. Also cares about checking if falling termino can be moved in required direction.

Termino.js

Class represents one termino with specific rotation. Once termino is sealed in the matrix at its final possition it is saved in PlayMatrix class.

Installation

Just download the ZIP unzip it and open game.html in your favorite browser.
